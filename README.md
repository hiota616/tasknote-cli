# TaskNote CLI

TaskNote CLI — консольное приложение для создания, просмотра, редактирования и удаления заметок.

Проект написан на Python. Управление происходит через интерактивное меню в терминале. Данные сохраняются локально в JSON-файл.

## Возможности

Программа позволяет:

- создавать заметки;
- просматривать все заметки;
- фильтровать заметки по статусу;
- редактировать заголовок, описание, дедлайн и статус;
- удалять заметки с подтверждением;
- хранить данные в локальном файле `notes.json`.

## Статусы заметок

У заметки может быть один из трёх статусов:

```text
in_process
done
archived
```

## Установка и запуск

### 1. Скачать проект

Склонируйте репозиторий:

```bash
git clone https://github.com/hiota616/tasknote-cli.git
```

Перейдите в папку проекта:

```bash
cd tasknote-cli
```

Вместо `USERNAME` укажите имя GitHub-аккаунта владельца репозитория.

---

### 2. Создать виртуальное окружение

#### Windows

```bash
python -m venv .venv
```

Активировать виртуальное окружение:

```bash
.venv\Scripts\activate
```

#### macOS / Linux

```bash
python3 -m venv .venv
```

Активировать виртуальное окружение:

```bash
source .venv/bin/activate
```

---

### 3. Установить зависимости

После активации виртуального окружения установите зависимости из файла `requirements.txt`:

```bash
pip install -r requirements.txt
```

В проекте используется внешняя библиотека `InquirerPy` для интерактивного меню в консоли.

---

### 4. Запустить программу

```bash
python main.py
```

После запуска откроется главное меню программы.

## Главное меню

После запуска доступны действия:

```text
Create a new note
View notes
Delete the note
Edit an existing note
Close the program
```

## Создание заметки

При создании заметки пользователь вводит:

- заголовок;
- текст заметки;
- дедлайн.

Если дедлайн не нужен, поле можно оставить пустым.

Каждая заметка получает уникальный ID.

## Просмотр заметок

В режиме просмотра можно:

- показать заметки со статусом `in_process`;
- показать заметки со статусом `done`;
- показать архивные заметки;
- показать все заметки.

## Редактирование заметки

При редактировании можно изменить:

- заголовок;
- описание;
- дедлайн;
- статус.

Перед сохранением изменений программа запрашивает подтверждение.

## Удаление заметки

При удалении программа показывает список заметок.

В списке отображаются:

- заголовок;
- часть описания;
- статус;
- короткий ID заметки.

Перед удалением программа запрашивает подтверждение.

## Хранение данных

Заметки сохраняются в файл:

```text
notes.json
```

Файл создаётся автоматически при первом запуске программы.

Данные хранятся в JSON-формате. Каждая заметка хранится по уникальному ID.

Пример структуры:

```json
{
    "note_id": {
        "title": "Example title",
        "text_note": "Example note text",
        "created_at": ["2026-07-13", "12:30:00"],
        "updated_at": null,
        "deadline": null,
        "status": "in_process"
    }
}
```

## Особенности реализации

В проекте используются:

- работа с файлами через модуль `os`;
- чтение и запись JSON через модуль `json`;
- генерация уникальных ID через модуль `uuid`;
- работа с датой и временем через `datetime`;
- интерактивное меню через библиотеку `InquirerPy`;
- обработка ошибок при чтении и записи файла;
- безопасная запись данных через временный файл;
- проверка пустого пользовательского ввода;
- подтверждение перед удалением и редактированием.
